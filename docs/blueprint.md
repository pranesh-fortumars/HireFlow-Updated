# **App Name**: HireFlow Glass

## Core Features:

- Triple-Tier Access Control: Secure, role-based login system for Admin, HR Executive, and Interviewer with protected routes and state-persistent sessions.
- Glassmorphism Applicant Hub: An interactive management interface for creating and tracking candidate profiles with a premium frosted-glass aesthetic.
- Dynamic Interview Coordinator: Seamlessly assign interviewers and schedule slots with real-time availability updates powered by LocalStorage state synchronization.
- Personalized Interviewer Portal: Tailored view for interviewers to access only their assigned candidates and submit structured evaluation feedback.
- Executive Analytics Dashboard: Comprehensive reporting suite featuring animated Recharts to track application funnel metrics and rejection rates.
- Global Notification System: Fluid, toast-style notifications powered by Framer Motion to alert users of scheduling changes and form submissions.

## Style Guidelines:

- Primary color: Deep Vivid Indigo (#6366F1) for a high-end SaaS feel. Background: Ultra-dark desaturated Navy (#0B0C14) to enhance the neon-glass glow effects.
- Accent colors: Electric Cyan (#06B6D4) for primary actions and Vibrant Purple (#A855F7) for interactive hover states.
- Font pairing: 'Space Grotesk' for bold, tech-forward headlines and 'Inter' for highly legible, objective body text.
- Sophisticated grid-based dashboard using backdrop-blur-xl cards and a slim, persistent sidebar navigation for optimal multitasking.
- Heavy use of Framer Motion for staggered list entrance, smooth layout transitions, and subtle floating background particles to suggest depth.
- Thin-stroke line icons with glow effects and color gradients consistent with the Indigo-Cyan primary palette.
